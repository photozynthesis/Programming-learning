## 1. IntelliJ IDEA

### 1.1 快捷键

- ctrl + space				代码提示
- ctrl + d 					复制一行
- ctrl + y 					删除一行
- ctrl + alt + L			        格式化
- ctrl + /				        单行注释
- ctrl + shift + /			        多行注释
- alt + shift + 方向		        上下移动当前行

### 1.2 快捷操作

- array.fori

  一键遍历数组

- 5.fori

  一键固定次数循环

- ctrl + alt + 下                          复制一行
